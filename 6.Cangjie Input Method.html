Cangjie Input Method（仓颉输入法）

Background

After returning to Taiwan in 1973, Zhu Fengfu began to improve Chinese typewriters and research Chinese retrieval methods. After three years of research, he published the "Shape Contour Coding Method" in 1976, using 52 Chinese characters as search letters, and dividing each Chinese character into three letters for coding.

In 1978, Zhu and his assistant Shen Honglian rewrote the stemming method, reducing the number of searched letters from 52 to 26 to fit on an ordinary computer keyboard. In 1979, he successfully implemented a Chinese communication system using Chinese characters as internal codes.

He continued to refine the shape contour encoding method and in 1979 founded Zero One to develop the second generation method. In 1982, he selected 40,000 characters from the "Kangxi Dictionary", improved the second generation, and published it as the third generation.

At present, font coding is the most widely used traditional Chinese character input method. The repetition rate is low and the font library is large. However, due to the emphasis on Chinese character retrieval, a complete "axis system" that Zhu Chu thought could solve the six major problems of Chinese characters has not been fully realized.

Features

  • Cangjie relies on a set of 24 basic keystrokes that correspond to various character components, such as horizontal lines, vertical lines, and curves.
  • The method assigns a unique code to each Chinese character based on its structure, allowing users to input characters quickly and accurately.
  • It supports both Traditional and Simplified Chinese characters, but it is primarily used for Traditional Chinese characters.

    • Usage method

    Instead of typing phonetics like Pinyin, users type the shapes of the characters. Each key on the keyboard represents a basic shape or stroke, and characters are input by combining these shapes.

    To use Cangjie, you need to learn the 24 basic Cangjie radicals and their corresponding keys. For example:

    1. 日 (sun)
    2. 月 (moon)
    3. 金 (metal)
    4. 木 (wood)
    5. 水 (water)

    To type a character, break it down into its radicals and type the corresponding keys. For instance, to input "明" (meaning bright), you would type "AB" since it consists of the radicals 日 (A) and 月 (B).

    • Pros and cons

    Advantages
    1. High accuracy: Due to the unique code assigned to each character, the Cangjie Input Method offers high accuracy and reduces the likelihood of mistyping.
    2. Speed: Once users become proficient, they can type quickly and efficiently.
    3. Learning tool: Learning the Cangjie Input Method can help users better understand the structure and composition of Chinese characters.

    Disadvantages
    1. Steep learning curve: The Cangjie Input Method requires users to learn and memorize a significant number of codes, making it challenging for beginners.
    2. Limited use: Although it supports both Traditional and Simplified Chinese characters, the Cangjie Input Method's primary user base is in regions that use Traditional Chinese characters, such as Taiwan and Hong Kong.
