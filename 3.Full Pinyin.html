Full Pinyin

Full Pinyin(全拼) input method is a Chinese character input method based on the Romanization of the pinyin system, using the Latin alphabet to represent the pronunciation of Mandarin.

  • Background
  • Experience
  • Advantages

Background

The earliest Chinese input method is Pinyin input method. However, the initial Pinyin input method had problems such as fixed word order, no support for phrase and whole sentence input, which made input very inefficient. With the development of computer technology, the function of Pinyin input method was improved, and after 1990s, Pinyin input method software started to support phrase input and whole sentence input.

Full Pinyin input method is a kind of Pinyin input method, which maps the complete pinyin of each Chinese character to the keyboard and forms words and sentences by continuously inputting the letters corresponding to the pinyin of the Chinese character.

Experience

Full Pinyin input method uses common computer or mobile phone keyboard, including number keyboard, letter keyboard and function keys, generally QWERTY keyboard. To use Full Pinyin input method, users need to enter the full pinyin of each Chinese character on the keyboard, and then follow the prompts to select the correct word or candidate.

Take the mobile phone keyboard as an example. To type "中文"（Pinyin: zhongwen）, we need to type "ZHONGWEN" in keyborad.


But some input methods have memory function. Therefore, in general, when we type "中文" once with the full spelling, we can type "中文" again without typing the full spelling.


Advantages

The advantages of Full Pinyin input method are as follows:

  1. Easy Input: Full Pinyin input method maps the full pinyin of each Chinese character to each letter on the keyboard, which makes Full Pinyin input method simple and can reduce the probability of error.
  2. Word Diversity: Since Full Pinyin input method needs to input all the complete pinyin of each Chinese character, there are relatively few differences and changes between pinyin. So, Full Pinyin input method can generate more words and phrases, covering more language expressions.
  3. Easy to Learn & Suitable for Teaching: As one of the mainstream Chinese input methods, simple and intuitive, Full Pinyin input method is easy to learn and master. Compared with other input methods such as Wubi, it is more suitable for Chinese learners.
