Phonetic

The phonetic input method is a Chinese input method that uses phonetic symbols to input Chinese characters.

  • Background
  • Experience
  • Advantages

Background

Because the phonetic symbols are taught as the method of Chinese spelling in elementary schools of the Republic of China, it is one of the most popular Chinese input methods in Taiwan. As long as you know the pronunciation of the word, you can input characters by using this input method.

The keyway scheme of Phonetic input method

There are five key schemes for Phonetic input method, namely, Daqian phonetic keyboard, Yitian phonetic keyboard, IBM phonetic keyboard, Jingye phonetic keyboard, Xu's phonetic keyboard and phone number keypad.

Daqian phonetic keyboard

Daqian type is the most widely used input method in Taiwan. Microsoft Windows calls it the standard type, and it is its default keyboard arrangement.


Yitian 26-key phonetic keyboard

The Yitian 26-key keyboard is improved from the Yitian 41-key keyboard.

Yitian keyboard is the phonetic symbol arrangement used on Yitian computer, one of the earliest Chinese computers.

Xu's phonetic keyboard

The inventor is Prof. . The "Xu's keyboard" only uses 25 keys, and the "Q" key is reserved for switching input. The arrangement of the "Xu's Keyboard" is suitable for those who are familiar with the English keyboard, and it is also easy to realize touch typing.

Usage method

The phonetic notation for "中文" is "ㄓㄨㄥ ㄨㄣˊ". Take the Daqian phonetic input keyboard as an example, to input "中 (Pinyin: zhong)", you need to press the "5" key to represent "ㄓ", the "J" key" to represent "ㄨ", /" to represent "ㄥ", "space key " to represent "first tone"; input "文 (Pinyin: wen)", you need to press "J" key to represent "ㄨ", "P" key to represent "ㄣ", "6" key to represent "second sound" ( ˊ). So to input “中文”, we need to input “5J space bar JP6” in the phonetic input method.

For those who are familiar with Zhuyin, the Zhuyin input method can be used easily whether typing traditional or simplified characters. But for people who don't know Zhuyin, it's difficult.

It is not recommended for people who do not know phonetic symbols.(Only the author's own point of view)

Advantages

  1. The Phonetic input method is easy for Taiwanese to get started. Almost everyone in Taiwan knows the phonetic symbols and can use them without special study.
  2. Compared with the Chinese Pinyin input method, the Phonetic input method can usually type out the sound of a word without following the pronunciation order, and has the function of quick replacement of phonetic symbols, finals, and tones, and usually supports parallel input.
